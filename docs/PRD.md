## 1. 專案概述

### 1.1 背景與動機

許多閱讀愛好者習慣在讀完書後記下心得與感受，但這些內容往往散落在便利貼、紙本筆記或多個 App 中，難以整理與查找。「讀書筆記本」系統旨在提供一個統一、結構化的數位空間，讓每一份閱讀足跡都能被妥善保存、分類與檢索，把零散的閱讀心得轉化為個人知識庫。

### 1.2 目標用戶

| 用戶類型 | 描述 |
|---|---|
| 一般讀者 | 有閱讀習慣、希望記錄讀書心得的一般大眾 |
| 學生 | 需要整理課外讀物或學術書目的學生 |
| 書評愛好者 | 喜歡為書籍評分並留下詳細評論的用戶 |

### 1.3 核心價值主張

> 讓每一本讀過的書，都留下一份有意義的數位印記。

- **記錄**：快速建立書籍筆記，不讓靈感與心得流失
- **分類**：透過標籤與分類功能，整理成個人書單知識庫
- **檢索**：隨時搜尋過去的閱讀記錄，找回珍貴的閱讀洞見

## 2. 功能需求

### 2.1 主要功能列表

#### 功能一：書籍記錄（新增 / 編輯 / 刪除）

作為**一般讀者**，我希望能**新增一本書的基本資訊（書名、作者、出版年份、封面）**，以便**建立我的個人書單**。

| 欄位 | 說明 |
|---|---|
| 書名 | 必填，最多 200 字元 |
| 作者 | 選填 |
| 出版年份 | 選填 |
| ISBN | 選填 |
| 封面圖片 URL | 選填 |

---

#### 功能二：閱讀心得記錄

作為**一般讀者**，我希望能**針對每本書撰寫閱讀心得與摘要**，以便**日後回顧書中重點與個人感受**。

| 欄位 | 說明 |
|---|---|
| 心得內容 | 支援純文字，可多行輸入 |
| 閱讀狀態 | 「想讀」/ 「閱讀中」/ 「已讀完」 |
| 閱讀開始 / 完成日期 | 選填 |
| 重點摘錄 | 可記錄書中金句或段落（選填）|

---

#### 功能三：評分系統

作為**一般讀者**，我希望能**對每本書進行 1 至 5 顆星的評分**，以便**快速標記書籍的好評程度並與自己的閱讀品味對照**。

| 項目 | 說明 |
|---|---|
| 評分範圍 | 1 星（差）～ 5 星（極好） |
| 顯示方式 | 星星圖示互動式選取 |
| 可修改 | 已評分可隨時更新 |

---

#### 功能四：分類與標籤管理

作為**一般讀者**，我希望能**為書籍設定分類（例如：心理學、小說、商業）並加上自訂標籤**，以便**快速瀏覽同類型的書籍並建立個人書單**。

| 項目 | 說明 |
|---|---|
| 分類 | 系統預設分類 + 使用者自訂 |
| 標籤（Tags）| 使用者自由設定，多選 |
| 過濾書單 | 可依分類 / 標籤篩選顯示 |

---

#### 功能五：全文搜尋

作為**一般讀者**，我希望能**透過關鍵字搜尋書名、作者或心得內容**，以便**在大量筆記中迅速找到我想查閱的書籍或段落**。

| 項目 | 說明 |
|---|---|
| 搜尋範圍 | 書名、作者、心得內容、標籤 |
| 搜尋方式 | 關鍵字模糊比對（LIKE） |
| 結果排序 | 依相關性或最近更新時間 |

---

#### 功能六：書單總覽儀表板

作為**一般讀者**，我希望能**在首頁看到我的閱讀統計與近期筆記**，以便**掌握自己的閱讀進度與習慣**。

| 項目 | 說明 |
|---|---|
| 已讀書籍數量 | 數值統計 |
| 平均評分 | 所有書籍平均星數 |
| 最近更新的筆記 | 前 5 筆 |
| 閱讀中書目 | 清單顯示 |

## 3. 非功能需求

### 3.1 技術限制

| 項目 | 規格 |
|---|---|
| 後端框架 | Python Flask |
| 模板引擎 | Jinja2 |
| 資料庫 | SQLite |
| 前端 | HTML + CSS（Vanilla）+ 少量 JavaScript |
| 部署環境 | 本機開發為主（localhost） |

### 3.2 效能考量

- 首頁與書單頁面載入時間應在 **2 秒以內**
- 搜尋結果應在 **1 秒以內** 回傳
- SQLite 資料庫適用於單人使用，不支援多人同時讀寫

### 3.3 安全考量

- 所有表單輸入需進行基本的 **XSS 防護**（Jinja2 預設 auto-escape）
- 使用 **CSRF Token** 保護 POST 請求（可使用 Flask-WTF）
- 資料庫操作使用 **參數化查詢**，防止 SQL Injection

### 3.4 可用性與介面

- 介面使用**繁體中文**
- 支援常見桌面瀏覽器（Chrome、Firefox、Edge 最新版）
- 具備基本的 **RWD 響應式設計**，支援平板瀏覽
